# 🌙Night Screen

当您在**夜间**使用手机时，Night Screen可以帮助您**减少屏幕亮度**，减少对眼睛的伤害。

软件完全采用**Material 3 (You)**设计风格。除全局Dialog外，**完全使用Jetpack Compose**开发。
