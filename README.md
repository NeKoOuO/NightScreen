## 💡主要功能

- 支持从系统下拉**任务栏开关启动**
- 支持调整**屏幕颜色**
- 支持屏幕**常亮**
- 支持**自动降低系统亮度**
- 支持运行时通知栏显示**常驻通知**
- ……
